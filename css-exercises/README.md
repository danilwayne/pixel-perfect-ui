## 🚀 Como usar

### Opção 1: Abrir no navegador diretamente

1. Abra o arquivo `ejercicio.html` no VS Code
2. Edite conforme os exercícios do guia
3. Clique com **botão direito** no arquivo → **"Open with Live Server"** (se tiver a extensão Live Server instalada)
4. Se não tiver Live Server, abra o arquivo diretamente no navegador:
   - Windows: clique duas vezes no arquivo
   - Ou copie o caminho absoluto e cole na barra de endereço

### Opção 2: DevTools para debug

1. Abra o navegador com o arquivo
2. Pressione **F12** (ou **Ctrl+Shift+I**)
3. Use a aba **Elements** para inspecionar
4. Edite estilos diretamente no painel **Styles**
5. Teste mudanças antes de aplicar no código

## 🎨 Dicas de DevTools

### Debug visual rápido:
```css
* {
    outline: 1px solid red !important;
}
```

### Inspecionar elemento:
1. Clique em **Selecionar elemento** (ícone de cursor no DevTools)
2. Clique no elemento desejado na página
3. Veja todas as propriedades CSS aplicadas

### Editar estilos em tempo real:
1. Clique na propriedade no painel Styles
2. Digite novo valor
3. Pressione Enter para aplicar
